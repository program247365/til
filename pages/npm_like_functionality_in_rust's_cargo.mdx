---
title: "NPM Like Functionality in Rust's Cargo"
date: 2021-10-11
tag: rust cargo
---

- `cargo install cargo-edit`
  - will give you
    - `cargo add`
    - `cargo rm`
    - `cargo upgrade`
    - commands in cargo!

Checkout [https://github.com/killercup/cargo-edit](https://github.com/killercup/cargo-edit) for more info.
