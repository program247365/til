---
title: "Curl to Create Jira Tickets"
date: 2021-12-20
tag: curl jira
---

```zsh
JIRA_INSTALL="YOUR URL HERE"

for ticket in OMS-86 OMS-109 OMS-8; do echo -n "$ticket... "; curl -d '{"update": {"labels": [{"add": "techdebt"}]}}' "${JIRA_INSTALL}/rest/api/2/issue/${ticket}" -H 'authorization: Basic <BASE64'd_IN_FORM_OF username:password>' -X PUT -H 'content-type: application/json' -s -w "%{http_code}\n"; done
```

```zsh
# output shows the ticket # and then the HTTP code (204 in this case means success)
OMS-86... 204
OMS-109... 204
OMS-8... 204
```
